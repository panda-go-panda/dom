Меня зовут Филоненко Нина. Я предприниматель с 19-летним стажем. Создано, продано, куплено много бизнесов. Естественно, получено много опыта и разного образования – управленческого, коучингового, дизайнерского.

Сейчас настал момент, когда сильно трансформировались смыслы – ради чего, для чего?

Совершенно точно, что уже не для того, чтобы заработать на кусок хлеба, совершенно точно, чтобы не доказывать свою «крутизну», но точно знаю, что для людей, которые хотят развиваться.

Для людей, которым нужен маленький толчок, чтобы начать делать великие дела, для людей, которым не достает веры в себя и, конечно, для людей, которые это осознают! Для детей, открытых новому, для наших родителей, которые, думают, что жизнь прошла, но на самом деле это Неправда.

Конечно, как настоящему предпринимателю, для меня важно, чтобы этот проект был рентабельным и выгодным для инвесторов, и моя ответственность перед партнерами и будущими сотрудниками не позволит, создавать другое.
